# DSA4262 Assignment 1 — Mental Health Data Visualisation

This repository contains the code, data, and notebook for DSA4262 Assignment 1, which explores suicide trends and post-psychiatric-discharge suicide risk using international public health datasets. The analysis follows a macro-to-micro narrative to support evidence-informed intervention design.

## Data Sources

- World Bank / WHO Suicide Mortality Rate  
  Indicator: SH.STA.SUIC.P5  
  Source: World Bank Data (derived from WHO estimates)  
  https://data.worldbank.org/indicator/SH.STA.SUIC.P5
  
- OECD Health Care Quality Indicators — Suicide Following Discharge  
  Indicator: Suicide within 30 days following discharge from psychiatric care  
  Source: OECD Health Statistics  
  https://data-explorer.oecd.org/vis?tm=suicide&pg=0&snb=6&df[ds]=dsDisseminateFinalDMZ&df[id]=DSD_HCQO%40DF_MH&df[ag]=OECD.ELS.HD&df[vs]=2.1&dq=.A...._T.OBS..&pd=2015%2C&to[TIME_PERIOD]=false  

## Files

- `notebooks/assignment1.ipynb` — Main analysis notebook with plots and narrative  
- `data/` — Raw datasets used for analysis
